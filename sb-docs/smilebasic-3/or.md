# OR

> **Category:** Bit Operations

Logical OR of Numerical value 1 and Numerical value 2 (Addition of bits)

## Format

```sb3
Variable=Numerical value 1 OR Numerical value 2
```

## Arguments

| Argument | Description |
| --- | --- |
| `Numerical value 1` | Bit string 1 |
| `Numerical value 2` | Bit string 2 |

## Examples

```sb3
A=128 OR &HA3
```
