# PRGDEL

> **Category:** Source code manipulation

Deletes the current line

## Format

```sb3
PRGDEL [Number of lines to delete]
```

## Arguments

| Argument | Description |
| --- | --- |
| `Number of lines to<br>delete` | - Number of lines to delete (If omitted, one line will be deleted)<br>- If a negative value is specified, all lines will be deleted |

## Examples

```sb3
PRGDEL
```
