# BGSTOP

> **Category:** BG

Stops BG animation

## Format

```sb3
BGSTOP [Layer]
```

## Arguments

| Argument | Description |
| --- | --- |
| `Layer` | Target layer number: 0-3<br>* If the layer is omitted, animation of all layers will be stopped. |

## Examples

```sb3
BGSTOP
```
