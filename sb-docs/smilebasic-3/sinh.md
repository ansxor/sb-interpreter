# SINH

> **Category:** Mathematics

Returns the hyperbolic sine value

## Format

```sb3
Variable = SINH( Numerical value )
```

## Arguments

| Argument | Description |
| --- | --- |
| `Numerical value` | Real-type number for which to find the hyperbolic sine |

## Return Values

Hyperbolic sine value found

## Examples

```sb3
A=SINH(1)
```
