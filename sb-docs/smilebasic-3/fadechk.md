# FADECHK

> **Category:** Screen control

Gets the state of the fading animation

## Format

```sb3
Variable=FADECHK()
```

## Return Values

```
TRUE= Animation in progress, FALSE= Animation suspended
```

## Examples

```sb3
R=FADECHK()
```
