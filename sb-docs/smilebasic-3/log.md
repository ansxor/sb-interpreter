# LOG

> **Category:** Mathematics

Finds the logarithm

## Format

```sb3
Variable = LOG( Numerical value [,Base ] )
```

## Arguments

| Argument | Description |
| --- | --- |
| `Numerical value` | Antilogarithm |
| `Base` | Base (* If omitted, the natural logarithm will be found.) |

## Return Values

Result found

## Examples

```sb3
A=LOG(2,2)
```
