# DEG

> **Category:** Mathematics

Finds the degree value from a radian value

## Format

```sb3
Variable = DEG( Numerical value )
```

## Arguments

| Argument | Description |
| --- | --- |
| `Numerical value` | Radian value |

## Return Values

Degree value found from the radian value

## Examples

```sb3
A=DEG(0.5*PI())
```
