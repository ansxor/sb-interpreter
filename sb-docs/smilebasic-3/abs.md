# ABS

> **Category:** Mathematics

Gets the absolute value

## Format

```sb3
Variable = ABS( Numerical value )
```

## Arguments

| Argument | Description |
| --- | --- |
| `Numerical value` | Numerical value for which to get the absolute value |

## Return Values

Absolute value

## Examples

```sb3
A=ABS(-12.345)
```
