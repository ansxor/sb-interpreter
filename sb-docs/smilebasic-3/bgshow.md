# BGSHOW

> **Category:** BG

Shows the BG screen

## Format

```sb3
BGSHOW Layer
```

## Arguments

| Argument | Description |
| --- | --- |
| `Layer` | Target layer number: 0-3 |

## Examples

```sb3
BGSHOW 0
```
