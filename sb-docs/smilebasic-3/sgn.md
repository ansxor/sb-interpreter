# SGN

> **Category:** Mathematics

Gets the sign

## Format

```sb3
Variable = SGN( Numerical value )
```

## Arguments

| Argument | Description |
| --- | --- |
| `Numerical value` | Numerical value for which to get the sign |

## Return Values

0 or ±1

## Examples

```sb3
A=SGN(12.345)
```
