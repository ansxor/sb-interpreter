# RAD

> **Category:** Mathematics

Finds the radian value from a degree value

## Format

```sb3
Variable = RAD( Numerical value )
```

## Arguments

| Argument | Description |
| --- | --- |
| `Numerical value` | Degrees: 0-360 |

## Return Values

Radian value found from the degree value

## Examples

```sb3
R=RAD(45)
```
