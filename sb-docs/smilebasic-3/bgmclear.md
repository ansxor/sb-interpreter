# BGMCLEAR

> **Category:** Sound

Clears a user-defined piece of music

## Format

```sb3
BGMCLEAR [User-defined tune number]
```

## Arguments

| Argument | Description |
| --- | --- |
| `User-defined tune<br>number` | Tune number: 128-255 (If omitted, all defined tunes will be cleared) |

## Examples

```sb3
BGMCLEAR
```
