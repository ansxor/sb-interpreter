# TO

> **Category:** Basic instructions (control and branching)

Specifies the loop count end value

- See Comment for the FOR instruction for details regarding FOR to NEXT

## Format

```sb3
TO End value
```

## Examples

```sb3
FOR I=0 TO 9 STEP 2
 PRINT I;",";
NEXT
```
