# XOR

> **Category:** Bit Operations

Exclusive OR of Numerical value 1 and Numerical value 2 (If the values are the same, 0; if not, inversion)

## Format

```sb3
Variable=Numerical value 1 XOR Numerical value 2
```

## Arguments

| Argument | Description |
| --- | --- |
| `Numerical value 1` | Bit string 1 |
| `Numerical value 2` | Bit string 2 |

## Examples

```sb3
A=100 XOR &H4C
```
