# DELETE

> **Category:** Files

Deletes a file When run, a confirmation dialog will be displayed

## Format

```sb3
DELETE "[File type:]File name"
```

## Arguments

| Argument | Description |
| --- | --- |
| `File type:` | "TXT:" Texts and programs (optional)<br>"DAT:" Binary data (including graphics) |

## Examples

```sb3
DELETE "SAMPLE"
```
