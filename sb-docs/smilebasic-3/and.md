# AND

> **Category:** Bit Operations

Logical AND of Numerical value 1 and Numerical value 2 (Multiplication of bits)

## Format

```sb3
Variable=Numerical value 1 AND Numerical value 2
```

## Arguments

| Argument | Description |
| --- | --- |
| `Numerical value 1` | Bit string 1 |
| `Numerical value 2` | Bit string 2 |

## Examples

```sb3
A=200 AND &HE7
```
