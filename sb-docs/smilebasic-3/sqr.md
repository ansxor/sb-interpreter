# SQR

> **Category:** Mathematics

Finds the positive square root

## Format

```sb3
Variable = SQR( Numerical value )
```

## Arguments

| Argument | Description |
| --- | --- |
| `Numerical value` | Numerical value for which to find the square root |

## Return Values

Positive square root found

## Examples

```sb3
A=SQR(4)
```
