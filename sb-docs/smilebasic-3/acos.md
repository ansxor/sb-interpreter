# ACOS

> **Category:** Mathematics

Returns the arc cosine value

## Format

```sb3
Variable = ACOS( Numerical value )
```

## Arguments

| Argument | Description |
| --- | --- |
| `Numerical value` | -1.0 to 1.0 |

## Return Values

```
Arc cosine (radian) value found
```

## Examples

```sb3
A=ACOS(1)
```
