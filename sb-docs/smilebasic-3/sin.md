# SIN

> **Category:** Mathematics

Returns the sine value

## Format

```sb3
Variable = SIN( Angle )
```

## Arguments

| Argument | Description |
| --- | --- |
| `Angle` | Radian |

## Return Values

Value found

## Examples

```sb3
A=SIN( RAD(45) )
```
