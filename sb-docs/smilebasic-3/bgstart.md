# BGSTART

> **Category:** BG

Starts BG animation

## Format

```sb3
BGSTART [Layer]
```

## Arguments

| Argument | Description |
| --- | --- |
| `Layer` | Target layer number: 0-3<br>* If the layer is omitted, animation of all layers will be started. |

## Examples

```sb3
BGSTART
```
