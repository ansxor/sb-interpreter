# VAL

> **Category:** Operations on strings

Gets a numerical value from a character string

## Format

```sb3
Variable = VAL( "Character string" )
```

## Arguments

| Argument | Description |
| --- | --- |
| `Character string` | A character string representing a number (e.g., "123"), or a string variable |

## Return Values

Numerical value interpreted from the character string

## Examples

```sb3
A=VAL("123")
```
