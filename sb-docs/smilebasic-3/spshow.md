# SPSHOW

> **Category:** Sprites

Starts displaying a sprite If used before SPSET, an error will occur

## Format

```sb3
SPSHOW Management number
```

## Arguments

| Argument | Description |
| --- | --- |
| `Management number` | Management number of the sprite to display: 0-511 |

## Examples

```sb3
SPSHOW 43
```
