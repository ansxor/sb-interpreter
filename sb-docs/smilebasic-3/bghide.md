# BGHIDE

> **Category:** BG

Hides the BG screen

## Format

```sb3
BGHIDE Layer
```

## Arguments

| Argument | Description |
| --- | --- |
| `Layer` | Target layer number: 0-3 |

## Examples

```sb3
BGHIDE 0
```
