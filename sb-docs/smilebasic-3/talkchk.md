# TALKCHK

> **Category:** Sound

Checks the status of speech synthesis

## Format

```sb3
Variable=TALKCHK()
```

## Return Values

```
FALSE = Stopped, TRUE = Playing
```

## Examples

```sb3
RET=TALKCHK()
```
