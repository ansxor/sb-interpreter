# ASIN

> **Category:** Mathematics

Returns the arc sine value

## Format

```sb3
Variable = ASIN( Numerical value )
```

## Arguments

| Argument | Description |
| --- | --- |
| `Numerical value` | -1.0 to 1.0 |

## Return Values

```
Arc sine (radian) value found
```

## Examples

```sb3
A=ASIN(0)
```
