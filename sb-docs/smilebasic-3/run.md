# RUN

> **Category:** Instructions available only in DIRECT mode

Program execution DIRECT mode only

## Format

```sb3
RUN [Program SLOT]
```

## Arguments

| Argument | Description |
| --- | --- |
| `Program SLOT` | Specified program SLOT to be executed (0 to 3. If omitted, 0) |

## Examples

```sb3
RUN
RUN 1
```
