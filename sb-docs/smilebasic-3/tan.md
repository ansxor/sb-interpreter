# TAN

> **Category:** Mathematics

Returns the tangent value

```sb3
Variable = TAN( Angle )
```

## Arguments

| Argument | Description |
| --- | --- |
| `Angle` | Radian |

## Return Values

Value found

## Examples

