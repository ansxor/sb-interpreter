# POW

> **Category:** Mathematics

Exponentiates a value

## Format

```sb3
Variable = POW( Numerical value, Multiplier )
```

## Arguments

| Argument | Description |
| --- | --- |
| `Numerical value` | Numerical value to exponentiate |
| `Multiplier` | Exponentiation multiplier |

## Return Values

Exponentiation result

## Examples

```sb3
A=POW(1,4)
```
