# ROUND

> **Category:** Mathematics

Gets the integer part (by rounding off to the nearest whole number)

## Format

```sb3
Variable = ROUND( Numerical value )
```

## Arguments

| Argument | Description |
| --- | --- |
| `Numerical value` | Source numerical value |

## Return Values

Integer value after rounding off

## See Also

FLOOR: Round-down, CEIL: Round-up

## Examples

```sb3
A=ROUND(12.345)
```
