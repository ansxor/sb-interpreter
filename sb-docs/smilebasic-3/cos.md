# COS

> **Category:** Mathematics

Returns the cosine value

## Format

```sb3
Variable = COS( Angle )
```

## Arguments

| Argument | Description |
| --- | --- |
| `Angle` | Radian |

## Return Values

Value found

## Examples

```sb3
A=COS( RAD(45) )
```
