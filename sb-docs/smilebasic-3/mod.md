# MOD

> **Category:** Bit Operations

Gets the remainder of Numerical value 1 divided by Numerical value 2

## Format

```sb3
Variable=Numerical value 1 MOD Numerical value 2
```

## Arguments

| Argument | Description |
| --- | --- |
| `Numerical value 1` | Number (or expression) to divide |
| `Numerical value 2` | Number (or expression) to divide by (Dividing by zero gives an error) |

## Examples

```sb3
A=200 MOD 5
```
