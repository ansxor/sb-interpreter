# SWAP

> **Category:** Basic instructions (variables and arrays)

Swaps the values of two variables Replacing a character string with a numerical value, or vice versa, is not allowed

## Format

```sb3
SWAP Variable 1, Variable 2
```

## Arguments

| Argument | Description |
| --- | --- |
| `Variable 1` | Variable to be replaced |
| `Variable 2` | Variable to be replaced with |

## Examples

```sb3
SWAP A,B
```
