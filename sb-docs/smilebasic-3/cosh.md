# COSH

> **Category:** Mathematics

Returns the hyperbolic cosine value

## Format

```sb3
Variable = COSH( Numerical value )
```

## Arguments

| Argument | Description |
| --- | --- |
| `Numerical value` | Real-type number for which to find the hyperbolic cosine |

## Return Values

Hyperbolic cosine value found

## Examples

```sb3
A=COSH(1)
```
