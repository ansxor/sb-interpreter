# BGCLR

> **Category:** BG

Clears the BG screen

## Format

```sb3
BGCLR [Layer]
```

## Arguments

| Argument | Description |
| --- | --- |
| `Layer` | Target layer number: 0-3 (If omitted, all layers) |

## Examples

```sb3
BGCLR
```
