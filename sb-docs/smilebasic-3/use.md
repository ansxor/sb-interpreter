# USE

> **Category:** Files

Makes a program in the specified program SLOT executable

## Format

```sb3
USE Program SLOT
```

## Arguments

| Argument | Description |
| --- | --- |
| `Program SLOT` | 0-3: Program SLOT |

## Examples

```sb3
USE 2
```
