# EXP

> **Category:** Mathematics

Exponentiates e (natural logarithm base)

## Format

```sb3
Variable = EXP( [ Numerical value ] )
```

## Arguments

| Argument | Description |
| --- | --- |
| `Numerical value` | Exponent (* If omitted, e will be returned.) |

## Return Values

Exponentiation result

## Examples

```sb3
A=EXP(2)
```
