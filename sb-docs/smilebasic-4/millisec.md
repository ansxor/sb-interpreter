# MILLISEC

Returns the number of milliseconds passed since SmileBASIC 4 started.

## Syntax

```sbsyntax
MILLISEC OUT ms%
```

| Output | Description |
| --- | --- |
| `ms%` | Milliseconds elapsed since SmileBASIC 4 started. |

## Examples

```sb4
PRINT MILLISEC()
```
