# SPUNLINK

Unlink a child sprite from its parent.

## Syntax

```sbsyntax
SPUNLINK childID%
```

| Input | Description |
| --- | --- |
| `childID%` | ID of the child sprite. 0-4095. |

## Examples

```sb4
SPLINK 1,0
SPUNLINK 1
```
