# MAINCNT

Returns the number of frames passed since SmileBASIC 4 started.

## Syntax

```sbsyntax
MAINCNT OUT frames%
```

| Output | Description |
| --- | --- |
| `frames%` | Frames elapsed since SmileBASIC 4 started. |

## Examples

```sb4
PRINT MAINCNT()
```
