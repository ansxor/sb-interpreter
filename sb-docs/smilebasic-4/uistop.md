# UISTOP

Stop the running UI program. Direct Mode only.

## Syntax

```sbsyntax
UISTOP
```

## Examples

```sb4
UISTOP 'stop the currently running UI program
```

## Notes

- The UI program is normally where the software keyboard runs, so do not run this command if you don't have a way to start it again, or you will have to restart SB4.
