# SUBSTOP

Stop the running subprogram. Direct Mode only.

## Syntax

```sbsyntax
SUBSTOP
```

## Examples

```sb4
SUBSTOP 'stop the currently running subprogram
```

## Notes

The running subprogram can also be stopped with the normal Break/Force Break buttons/keys.
