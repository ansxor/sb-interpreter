# M0 ‚Äî Scaffolding & spec pipeline (PRD)

> Status: ‚úÖ **done** ¬∑ Depends on: ‚Äî ¬∑ Read `prd/README.md` first.

This PRD is a **retrospective**: it documents what already exists so downstream agents
know the ground they're building on. Don't re-do this; extend it.

## Goal (met)

A buildable Rust workspace (native + wasm), the spec system seeded from the docs, a
two-phase conformance harness skeleton, and a deterministic CI gate.

## What exists

### Rust workspace (`crates/`)
- `sb-core` ‚Äî language core. **Implemented:** `error::ErrNum` (official table 3..47 +
  messages, tested), `value::{Value, ValueType}` (Int/Double/Str + coercion + truthiness,
  tested). **Stubbed for M1:** `lexer`, `parser`, `compiler`, `vm`, `builtins` (module
  files with design-reference docs).
- `sb-render` ‚Äî `Framebuffer` (RGBA8888) + faithful color expansion `expand5`/
  `rgb555_to_argb8888` (proven: `#WHITE=&HFFF8F8F8` ‚áí 5-bit expands by `<<3`, low bits
  zero). Screen dims: top 400√ó240, bottom 320√ó240.
- `sb-audio`, `sb-platform-native` (bin `sb`), `sb-platform-wasm` (cdylib), `sb-spec`
  (spec loader + `sb-spec-coverage` bin) ‚Äî scaffolded.

### Spec system (`spec/`) ‚Äî single source of truth
- `instructions/*.yaml` ‚Äî **248** instruction specs at `confidence: documented`,
  auto-generated by `tools/gen_specs.py` from `sb-docs/`.
- `reference/{errors,sysvars,constants}.yaml` ‚Äî 45 errors, 24 sysvars, 79 constants
  (with exact `bits`).
- `tests/*.yaml` ‚Äî hand/oracle test overlay (showcase: `floor.yaml`). Merged by `sb-spec`.
- `SCHEMA.md` documents the format + the two-layer (generated vs overlay) design.

### Harness (`harness/`) ‚Äî see `harness/README.md`
- `oracle/` (citra_rpc, extdata, framebuffer, audio), `diff/` (replay = Phase-B gate,
  run = 3-way differential), `fuzz/` (seeded generator), `harvest/` (writes committed
  fixtures), `corpus/` (ported `otya_test.sb3` golden + `cases/basics.yaml`).

### Tooling (`tools/`)
- `citra.py` (emulator RPC), `extract_code.py` (ROM `.code` extractor), `gen_specs.py`
  (docs ‚Üí spec).

### CI (`.github/workflows/`)
- `ci.yml` ‚Äî fmt, clippy `-D warnings`, build (native + wasm32), test, **spec-drift
  check**, coverage artifact. Hermetic.
- `harvest.yml` ‚Äî manual-only placeholder (no emulator in CI).
